About the Project

Smart Water Quality Monitoring System

This CBTP (Community-Based Technology Project) aims to develop a real-time water quality monitoring system using IoT technology. The system tracks pH, Temperature, and Turbidity using an Arduino Mega and broadcasts data via Bluetooth for immediate analysis.
